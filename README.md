City Guide

City Guide is a modern web application designed to help users explore cities and access useful information through a clean, simple, and user-friendly interface. The project focuses on responsive design, structured code, and modern frontend development practices.

Features

City-wise information display

Clean and responsive user interface

Dynamic content rendering

Well-organized project structure

Easy navigation between different sections

Tech Stack

TypeScript

JavaScript

HTML5

CSS3

Project Structure

The project follows a clean and modular structure with separate folders for source code, public assets, styles, and configuration files, making it easy to understand and maintain.

Getting Started

To run the project locally:

Clone the repository from GitHub.

Navigate to the project directory.

Install the required dependencies.

Start the development server.

Purpose of the Project

This project was created for learning and practice purposes to improve frontend development skills, understand modern project structuring, and work with web technologies such as TypeScript, HTML, and CSS.
